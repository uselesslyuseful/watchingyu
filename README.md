# WatchingYu
An unofficial fansite dedicated to the singer Hua Chenyu.
## Usage
The site can be accessed through the Github Pages linked to this repository. Most web browsers should be able to run the site without issue, although there is currently no mobile support. Any bugs can be reported to the email on the site. 
## Features
- Discography page containing all of Hua Chenyu's albums, contained singles, and lyrics for all singles that were not released as part of an album.
- Updates page containing Hua Chenyu's recent activities/events attended
- Home page with an introduction of Hua Chenyu and linked socials
- Music player that allows choice of song to play out of a curated selection
